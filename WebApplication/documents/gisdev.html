Adaption of Lane Model in ArcGIS Pro

In adapting the lane model to 3D for the Donnersberger Bridge surrounding area in Munich, the utilization of .laz, .las, and .gml datasets within ArcGIS Pro proves instrumental. By integrating these datasets, we can harness the rich geospatial information they provide, such as precise elevation data from LiDAR (.laz/.las) and semantic information from GML (.gml) files.

Non-classified point cloud data of the surrounding area in ArcGIS Pro. CityGML LoD2 Level Buildings of the surrounding area in FME Data Inspector.

Leveraging ArcGIS Pro's robust geoprocessing capabilities, we can seamlessly incorporate these datasets to refine the lane model, ensuring its accurate representation in three dimensions. During this process, non-classified point cloud data has been used, but with some deep learning supervised classification methods like ground classifier and tree classifier, we were able to extract the necessary features to create a smooth surface from the point cloud.

Point cloud data after deaplearning supervised classification methods in ArcGIS Pro. Ground class of the lane model of Donnersberger Bridge in ArcGIS Pro.

This refined surface data was then converted into a raster format, enabling the creation of a high-resolution Digital Surface Model (DSM) of the area, which proved invaluable for the adaptation process.

Created DSM of the surrounding from supervised classified point cloud data.

Pixel Editor Tools can be used on the DSM of the bridge to avoid voids and blanks on the surface.
